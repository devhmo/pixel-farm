# PixelFarm - Account Pixels Tracker

A pixel tracking application built with Next.js + React.

## Features

- **Real-time Tracking** — Live progress bars and countdown timers
- **Folder Organization** — Group accounts into folders
- **IP Tracking** — Track IP addresses per account with history
- **Full Customization** — Dark/Light mode, 6 progress bar styles, card customization
- **Import & Export** — JSON backup with merge/replace options
- **Notifications** — Browser notifications when accounts complete
- **i18n** — English, Russian, Portuguese (BR), Arabic (RTL)
- **PWA Support** — Install as app, works offline
- **Keyboard Shortcuts** — Quick actions via keyboard

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
```

Static export will be in the `out/` directory.
